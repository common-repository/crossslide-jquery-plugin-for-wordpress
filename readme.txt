== Description ==

** this plugin is no longer being update. Please feel free to adopt me! **




The CrossSlide jQuery plugin for WordPress is designed to quickly add the JS and CSS requirements to operate the CrossSlide (http://tobia.github.com/CrossSlide/) jQuery slideshow function to a website which includes an easy, clever static cross fade for WordPress.

The plugin allows users to control the speed, angle and transition feed from an easy to use control panel as well as upload images directly to WordPress (or select from existing uploads).

Japanese, German and French language support.

== Installation ==

To install the plugin, please upload the folder to your plugins folder and active the plugin. Once activated, add the php code &lt;?php cr_wp_crossslide(); ?&gt; to your website theme.
